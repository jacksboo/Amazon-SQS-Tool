Amazon-SQS-Tool
===============

Python tool for manipulation queues on Amazon SQS using JSON format messages.

Requirements
------------

- Python 2.7.x
- boto


Instructions
------------

- Copy ``config.ini.bkp`` to ``config.ini``.
- Set values ``queue``, ``access_key_id`` and ``secret_access_key`` on ``config.ini``.
- Run ``python main.py``.
